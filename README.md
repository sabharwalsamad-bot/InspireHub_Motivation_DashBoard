InspireHub is a simple and responsive React-based Quote Generator that fetches random inspirational quotes and allows users to like and save their favorite quotes.

🚀 Features
	•	Fetch random quotes from an API
	•	Like / Unlike quotes
	•	Persistent storage using localStorage
	•	Display total liked quotes
	•	Loading state handling
	•	Clean and minimal UI
	•   Filter Button

🛠️ Built With
	•	HTML
	•	CSS
	•	JavaScript (ES6)
	•	React (useState, useEffect)

📌 How It Works
	•	On load, the app fetches a random quote.
	•	Users can generate new quotes.
	•	Liked quotes are saved in the browser and remain even after refresh.
	•   Unique filter Button to filter out quotes of a particular author.
